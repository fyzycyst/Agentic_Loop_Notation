# Safety Reporting

ALN describes agent-loop control flow. It does not grant permission, override human approval, or make a loop safe by itself.

Please open a GitHub issue if you find an ALN example or document that could encourage:

- unsafe authority boundaries
- destructive production changes without approval
- exposure of secrets, private data, or credentials
- contacting third parties without consent
- spending money or deploying systems without a human gate
- claiming completion without evidence

When reporting, include:

1. The file or example.
2. The risky wording or behavior.
3. The safer boundary, verifier, stop rule, or return artifact you recommend.

Do not include secrets, private data, tokens, credentials, or exploit details in public issues.
